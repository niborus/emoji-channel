# emoji-channel
Removes all Messages from one or More Discord-Chats, where the Message contains more then emojis.

You need the libarys discord and emoji

# install discord.py

Linux/OS X
python3 -m pip install -U discord.py

Windows
py -3 -m pip install -U discord.py

See: https://pypi.org/project/discord.py/

# install emoji

Linux/OS X
$ pip install emoji --upgrade

See: https://pypi.org/project/emoji/

# init main.py

Insert the Channel-ID as Integer in channels. Example:
  channels = [543861314253316565, 543861314053984327]
Import the Discord-Token in DISCORD_TOKEN. Example:
  DISCORD_TOKEN = "IJustSmashedTheKeyboard.oiuhDe5ui.JFIOEAWJF93W"
